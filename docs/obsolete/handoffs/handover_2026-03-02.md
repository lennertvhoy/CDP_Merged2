### Create Email Engagement Workflow in Tracardi

**Objective:** Create a workflow that processes email engagement events and updates profile scores.

**Access:**
- GUI: http://137.117.212.154:8787
- Login: admin@admin.com / <redacted>

**Workflow Requirements:**
1. **Name:** "Email Engagement Scorer"
2. **Trigger:** Events `email.opened` and `email.clicked` from source `resend-webhook`
3. **Logic:**
   - When `email.opened` → increment `stats.engagement_score` by 10
   - When `email.clicked` → increment `stats.engagement_score` by 25
   - If `stats.engagement_score > 50` → add tag "Highly Engaged"

**Steps:**
1. Log in to Tracardi GUI
2. Navigate to Workflows → New Workflow
3. Configure trigger:
   - Event Type: `email.opened`, `email.clicked`
   - Source: `resend-webhook`
4. Add "Update Profile" action:
   - Field: `stats.engagement_score`
   - Operation: Increment
   - Value: 10 (for open) or 25 (for click)
5. Add "If" condition:
   - Condition: `stats.engagement_score > 50`
6. Add "Add Tag" action (in If branch):
   - Tag: "Highly Engaged"
7. Save and activate workflow

**Testing:**
1. Send test email via Resend API
2. Open email in email client
3. Click a link in email
4. Verify in Tracardi:
   - Events received
   - Profile score updated
   - Tag appears when score > 50
